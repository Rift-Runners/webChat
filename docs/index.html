inSight Webchat

Definição

Chat em NodeJS acessível pela web a qualquer momento;

Envia e recebe mensagens de n usuários dentro da aplicação;

Usuário autenticado através do Firebase utilizando o padrão OAuth2.0;

Cada mensagem é gravada com seu usuário, conteúdo e data e está atualizando um banco de dados disponibilizado na nuvem.

Technology Stack

Server side - Node.js

Database - MongoDB

Client side - Vanilla JS e jQuery

IoT - Arduino Uno

Frameworks e Libs

Express (HTTP) e Socket.io (Tempo real)

mLab (DBaaS)

Heroku (PaaS)

Gulp, Browserify e Mongoose

Node-red, Node-red-contrib-firebase e Arduino IDE (C++)

Ambiente de Desenvolvimento

Desenvolvido e disponibilizado no Github

É necessário possuir as ferramentas: git, bash e nodejs (+npm)

Com o bash aberto, digite:

git clone https://github.com/Rift-Runners/webChat

Entre na pasta baixada e, ainda dentro do bash:

npm install

Ambiente de Desenvolvimento

Abra um novo bash e insira:

npm install -g node-red node-red-contrib-firebase

Por fim, você necessitará estar executando em dois bash(es) diferentes:

node-red

E:

gulp default-watch

Inicializando a aplicação localmente

Em bash(es) diferentes, entre na pasta do projeto e digite:

npm start

E:

node-red

A aplicação iniciará (por default) em localhost:8080 e os fluxos do node-red em localhost:1880
